# PipirkaTube 🎬

Платформа для видео, похожая на YouTube, где пользователи могут загружать видео, смотреть видео других пользователей, реагировать на них и иметь свои аккаунты.

## Возможности

- ✅ Регистрация и вход пользователей
- ✅ Загрузка видео с превью
- ✅ Просмотр видео
- ✅ Реакции на видео (лайки/дизлайки)
- ✅ Профили пользователей
- ✅ Современный и красивый UI

## Технологии

### Backend
- Node.js + Express
- SQLite (база данных)
- JWT (аутентификация)
- Multer (загрузка файлов)
- bcryptjs (хеширование паролей)

### Frontend
- React
- React Router
- Axios
- React Icons

## Установка

1. Установите все зависимости:
```bash
npm run install-all
```

2. Настройте переменные окружения (опционально):
```bash
cd backend
cp .env.example .env
# Отредактируйте .env при необходимости
```

3. Запустите приложение:
```bash
npm run dev
```

Это запустит:
- Backend сервер на `http://localhost:5000`
- Frontend приложение на `http://localhost:3000`

## API Endpoints

### Аутентификация
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `GET /api/auth/me` - Получить текущего пользователя

### Видео
- `GET /api/videos` - Получить все видео
- `GET /api/videos/:id` - Получить видео по ID
- `GET /api/videos/user/:userId` - Получить видео пользователя
- `POST /api/videos/upload` - Загрузить видео (требует аутентификации)
- `DELETE /api/videos/:id` - Удалить видео (требует аутентификации)

### Реакции
- `POST /api/reactions/:videoId` - Добавить/изменить реакцию
- `GET /api/reactions/:videoId` - Получить реакцию пользователя
- `GET /api/reactions/:videoId/counts` - Получить количество реакций

### Пользователи
- `GET /api/users/:userId` - Получить профиль пользователя
- `PUT /api/users/:userId` - Обновить профиль (требует аутентификации)

## Использование

1. Зарегистрируйте новый аккаунт
2. Войдите в систему
3. Загрузите видео через страницу "Загрузить"
4. Просматривайте видео на главной странице
5. Реагируйте на видео лайками/дизлайками
6. Просматривайте профили других пользователей

## Примечания

- Видео сохраняются локально в папке `backend/uploads/videos`
- Превью сохраняются в `backend/uploads/thumbnails`
- Максимальный размер видео: 500MB
- Максимальный размер превью: 5MB
- Поддерживаемые форматы видео: mp4, webm, ogg
- Поддерживаемые форматы изображений: jpeg, jpg, png, gif

## Разработка

